Quick is a java console-based chat application for sending and managing messages.
User can register and log in using validated usernames,passsword,and south African phone numbers
The program users 3 classes-LoginForm for authentication\validation,Messages 1_ for creating \managing messages ,and MESSAGETEST to control overall flow
Users can send ,stored,or discard messages , with the system checking recipient numbers  and enforcing a 250-character limit 
Each message gets a unique ID and hash , and is stored in JSON file using file handling.
